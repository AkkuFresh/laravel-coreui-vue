# Laravel CoreUI Vue

> Laravel + CoreUI + Vue Boilerplate

## How to Install
* Clone this repository
* Install depencies
```bash
$ composer install
$ npm install
```
* Copy `.env` file
```bash
$ cp -r .env.example .env
```
* Generate Application key
```bash
$ php artisan key:generate
```
* Add write permission
```bash
$ chmod -R go+w storage bootstrap/cache
```
